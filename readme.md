# StrongMemory 🧠

StrongMemory is a simple memory card game built with Python.  
The project is designed to practice core programming concepts such as:

- Object-Oriented Programming (OOP)
- Game loop logic
- State management
- Modular project structure
- Git workflow

---

## 📁 Project Structure
StrongMemory/
│
├── core/ # Game logic
├── entities/ # Game objects (Card, Player, etc.)
├── ui/ # Interface layer
├── assets/ # Images / resources
│
├── main.py # Entry point
├── settings.py # Configuration
├── requirements.txt
└── README.md
